# RoamVerse product strategy

## Brand

**RoamVerse** — *Travel, made more memorable.* The identity is quiet, luminous, and editorial: soft violet aurora light against near-black space. Its mark is a compact `R` as a portal, not a pin.

## Elevator pitch

RoamVerse is a team of AI travel specialists that turns a fleeting feeling into an experience your friends want to join, remix, and remember.

## Problem and audience

Planning tools optimize logistics, while inspiration tools leave people with tabs and indecision. RoamVerse serves taste-led explorers, friend groups, and creative professionals (ages 22–40) who value distinctive moments over exhaustive itineraries.

## Viral loop

An agent produces a beautiful “Journey Drop” → the creator shares it → friends react or remix it → the group co-authors the trip → their story becomes the next inspiration artifact.

## MVP feature set

1. Intent-to-Journey Drop (streaming coordinator response).
2. Specialist agent roster and transparent progress.
3. Journey canvas for timeline, budget, food, hidden-gem, packing, and map modules.
4. Shareable public story page with remix entry point.

## Primary journey

Discover → express a feeling → watch the team think → receive a Journey Drop → invite collaborators → refine modules → publish a story.
