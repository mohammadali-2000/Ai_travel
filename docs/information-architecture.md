# Information architecture

```text
RoamVerse
├── Workspace / dashboard
│   ├── Create journey
│   ├── Agent activity
│   └── Saved drops
├── Journey canvas
│   ├── Timeline
│   ├── Map
│   ├── Budget
│   ├── Food
│   ├── Hidden gems
│   └── Packing
├── Story / share
├── Profile
└── Settings
```

The journey canvas is the application’s source of truth. Every module is an agent-backed view of the same journey, rather than a separate planning product.
